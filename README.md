# Online Exam Detection using SVM

## Objective
This project detects suspicious behavior during online examinations using Machine Learning.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Algorithm Used
Support Vector Machine (SVM)

## Files Included
- Project Notebook
- Report
- PPT

## Output
The model predicts suspicious and non-suspicious activities.
